Creativity Workshops

Explore Procedural Art, Animation, 3D Modeling, and More!

We offer summer reativity workshops online, to help people learn how to create amazing generated art using the computer! Using special technology allows us to express ourselves in different ways. Using games and interactive classroom projects we help people to discover the world of technology. This course is available for all ages. The course includes two, one hour workshops and a class project in between. A desktop computer with internet access and ability to use Google Chrome browser is required.
